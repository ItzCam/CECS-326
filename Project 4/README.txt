Hello! To run our program simply follow these steps!

1. In that same directory where the code is located open the terminal, and input the following commands

	ls -> this is to list everything is in the directory

	make fcfs-> this will build the fcfs scheduler
	make priority -> this will build the priority scheduler 
	make rr -> this will build the round robin scheduler 

2.To run the code, input the following
	for fcfs: [ ./fcfs schedule.txt]
	for priority: [ ./priority pri-schedule.txt]
	for rr: [ ./rr rr-schedule.txt]

	this will execute the scheduler
